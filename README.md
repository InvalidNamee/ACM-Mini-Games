# 社团活动小游戏静态站

一个基于 `Vue 3 + Vite + TailwindCSS` 的静态网页项目，适合社团活动现场使用。

## 已实现内容

- 三个小游戏，并拆成独立路由
- 总积分统计与奖励档位判定
- 固定奖励阈值常量：`打卡盖章`、`奖品 1`、`奖品 2`
- 本地浏览器持久化成绩
- 单页静态站结构，适合部署

## 小游戏说明

1. `二分答案`
   系统在指定范围内随机生成整数，用户输入答案后会收到偏大或偏小的提示。`ceil(log2(范围大小))` 步内为满分，超出后逐步扣分。
2. `冒泡排序`
   用户只能选择相邻的两个数字进行交换，目标是把序列排成升序。逆序对数视为理论最优步数，超出后逐步扣分。
3. `四层汉诺塔`
   用户把四层圆盘从起始柱移动到目标柱。理论最优解为 15 步，超出后逐步扣分。

## 运行方式

```bash
npm install
npm run dev
```

## 打包静态文件

```bash
npm run build
```

构建产物默认在 `dist/` 目录，可直接部署到静态托管平台。
