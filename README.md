# 玩游戏，拿奖励！

这是一个基于 `Vue 3 + Vite + TailwindCSS` 的社团活动小游戏站点，适合现场展示、摊位互动和扫码体验。

## 项目特点

- 首页聚合全部小游戏，入口清晰
- 三个小游戏独立页面，方便单独游玩和分享
- 支持本地保存成绩，刷新后也能保留
- 按总分解锁不同奖励档位
- 纯前端静态站点，部署简单

## 游戏列表

- `二分答案`：根据提示猜出随机数字，尽量少猜几次
- `冒泡排序`：只能交换相邻数字，把序列排成升序
- `四层汉诺塔`：把四层圆盘搬到右侧，尽量用更少步数完成

## 奖励说明

项目会根据总分显示当前奖励档位。奖励档位和阈值都在 `src/config/gameRules.js` 中配置，可以按活动需要自行调整。

## 本地运行

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run dev
```

## 打包发布

```bash
npm run build
```

构建完成后，静态文件会输出到 `dist/` 目录，可直接部署到静态托管服务。

## 项目结构

```text
src/
  components/   # 三个小游戏组件
  composables/   # 成绩统计与本地存储逻辑
  config/       # 游戏与奖励配置
  views/        # 首页、页面壳和游戏入口页
```

## 说明

如果你要把它改成别的活动主题，通常只需要调整首页文案、`src/config/gameRules.js` 里的奖励配置，以及各个小游戏的展示文字。
